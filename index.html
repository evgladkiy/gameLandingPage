About

Description

Silly zombies is a game in the genre of mixture of runner and slasher, the condition of victory in which is to survive, kill the Boss and as many zombies as possible

Technologies

This game is built using: JavaScript, HTML5, CSS3, Web Audio API

Difficulty

You can choose the difficulty suitable for your level of play: Baby Mode, Normal Mode or Hell Mode
